Project Overview:
Our team has chosen to work on the Recipe Buddy project.
We have it all hosted on Andy Mahoney's server. That is, Fortrash.com/gOatmeal.

The goal of the project is to make an easy to use user friendly cooking interface for home chefs. Some of the special features
of this project include infinite scrolling, an interactive shopping cart, timers for the recipes, and search functionality with
our database.
